# Retail Sales MIS Dashboard (Excel)

##  Project Overview
This project is a **Retail Sales MIS Dashboard** developed using **Microsoft Excel** to analyze sales performance and support business decision-making.

The dashboard provides a consolidated view of **sales trends, regional performance, category contribution, and brand analysis** using interactive visuals and slicers.

---

##  Key Features
- KPI Cards: Total Sales, Total Units, YTD Sales, MAT Sales
- Interactive slicers for **Year, Category, and Region**
- Monthly Sales Trend analysis (2023–2024)
- Sales distribution by Region
- Category-wise revenue analysis
- Top 10 Brands performance
- Executive-level **Key Business Insights** section

---

##  Tools & Skills Used
- Microsoft Excel
- Pivot Tables & Pivot Charts
- GETPIVOTDATA
- Slicers & Report Connections
- Dashboard design using Excel shapes
- Business insight interpretation

---

##  Key Business Insights
- Total Sales reached **987M** with **5M units sold**
- **North region** is the highest revenue contributor
- **Soap and Lotion** are the top-performing categories
- **BrandD** leads overall brand performance
- Sales show stable trends with seasonal fluctuations
